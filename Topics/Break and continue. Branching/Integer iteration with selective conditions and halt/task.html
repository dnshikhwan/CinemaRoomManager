Write a Java program that takes an integer as input, and iterates over all integers from 1 up to the given number (both inclusive). When the number is divisible by 5, your program should continue to the next number without printing anything. For all other numbers, print the number, but if the number is divisible by 7, then stop the iteration and print the number followed by ': stopped' immediately. Let's implement it.


Sample Input:
6

Sample Output:
1
2
3
4
6


Sample Input:
12

Sample Output:
1
2
3
4
6
7: stopped



Memory limit: 256 MB
Time limit: 5 seconds
